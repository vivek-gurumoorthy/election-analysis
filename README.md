# election-analysis

## Project Overview
A Colorado Board of Elections Employee has given you the following tasks to complete the election audit of a recent local congressionial election.

1. Calculate the total number of votes cast
2. Get a complete list of candidates who received votes
3. Calculate the total number of votes each candidate received
4. Calculate the percentage of votes each candidate won
5. Determine the winner of the election based on popular vote

## Resources
- Data Source: election_results.csv
- Software: Python 3.6.1, Visual Studio Code 1.38.1

## Summary
The analysis of the election show that:
* There were "x" votes cast in the election
* The candidates were:
    * Candidate 1
    * Candidate 2
    * Candidate 3
* The candidate results were:
    * Candidate 1 received % of the vote and total votes
    * Candidate 1 received % of the vote and total votes
    * Candidate 1 received % of the vote and total votes
* The winner of the election was:
    * Candidate who received % of vote and total votes
